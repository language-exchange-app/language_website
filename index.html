How It Works

Are you learning a language and want to practice with a native speaker from around the world? Then this app is for you. Are you also interested in dating your language partner? No problem, we offer this feature too. With TalkiDate each user is required to upload an introductory video, and the application presents to the user videos that correspond to profiles that match their language and dating preferences. For each video the user must decide whether to like it or not and if there is a match then they can start talking to that person.
